Hi, my name is Wiktor.

Open Source Enthusiast & Data Freedom Fighter 💻

👋 Hi. I'm Wiktor, but you can call me spageektti. I'm a student with a passion for open source and open data, and I believe that knowledge should be free and accessible to everyone 📚. I also like playing games, especially 🏎️ Super Tux Kart and 🎮 retro classics.

🎁 Contributions 🎁

I'm proud to have contributed to some amazing projects, including:

  • tldr-pages: 📝 Maintainer, organization member. Lot of contributions. The tldr-pages project provides simplified and community-driven man pages.
  • OpenStreetMap: 🗺️ Contributor. Many contributions. OpenStreetMap is a collaborative project to create a free, editable map of the world.
  • Wikipedia: 📖 Contributor. Some contributions. Wikipedia is a free, open encyclopedia that anyone can edit.
  • Wikidata: 🗃️ Contributor. Some contributions. Wikidata is a free, collaborative knowledge base of structured data.
  • Commons: 🖼️ Contributor. Some contributions. Commons is a free, open media repository that anyone can contribute to.
  • OpenFoodFacts: 🍔 Contributor. Some contributions. OpenFoodFacts is a free, open database of food products from around the world.
